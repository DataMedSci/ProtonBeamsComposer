Installation
============

Stable version ::

    pip install pbc

Latest unstable version, directly GIT repository, using::

    pip install setuptools versioneer
    pip install git+https://github.com/DataMedSci/ProtonBeamComposer.git

To unistall, simply use::

    pip uninstall beprof
